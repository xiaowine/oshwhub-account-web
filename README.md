# OSHWHub 用户搜索工具

一个基于 Vue 3 + TypeScript 开发的 OSHWHub 用户搜索工具，提供用户搜索和详细信息查看功能。

## 功能特点

- 🔍 支持用户名和昵称搜索
- 👥 支持模糊搜索和精确匹配
- 📋 可配置搜索结果数量（1-50）
- 👤 查看用户详细资料
- 📱 响应式设计，支持移动端

## 技术栈

- Vue 3
- TypeScript
- Vite
- CSS 变量实现主题定制
- Fetch API
